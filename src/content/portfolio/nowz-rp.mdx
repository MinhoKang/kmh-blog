## Technical Challenges & Solutions

### Performance Optimization

**Challenge**: Chat system INP reached 3217ms with heavy message loads and images.

**Solution**: Strategic React.memo implementation with custom comparison functions:

```typescript
const MessageItem = React.memo(
  ({ message, onRead, onReply }) => {
    // Component implementation
  },
  (prevProps, nextProps) => {
    return (
      prevProps.message.id === nextProps.message.id &&
      prevProps.message.readAt === nextProps.message.readAt &&
      prevProps.message.status === nextProps.message.status
    );
  }
);
```

**Result**: Reduced INP from 3217ms to 87ms (97% improvement)

### Next.js Caching Strategy

**Challenge**: Query string changes caused server components to re-execute, breaking TanStack Query cache.

**Solution**: Implemented advanced caching configuration with strategic cache management:

```typescript
// next.config.ts
export default {
  experimental: {
    staleTimes: {
      dynamic: 30,
      static: 180,
    },
  },
  // Custom cache configuration
};
```

**Results**:

- **Scripting performance**: 22-35% improvement across pages
- **Rendering performance**: 4-36% faster layout calculations
- **Total main thread time**: 16-33% reduction

### Cross-tab Communication

**Challenge**: Parent and student chats open in separate tabs needed real-time synchronization.

**Solution**: BroadcastChannel API for efficient cross-tab state sync:

```typescript
class CrossTabSync {
  private channel = new BroadcastChannel("chat-sync");

  syncMessageAcrossTabs(message: Message) {
    this.channel.postMessage({
      type: "MESSAGE_UPDATE",
      payload: { message },
    });
  }
}
```

### Dynamic Modal Management

**Challenge**: 10+ different modals in chat interface caused bundle size issues.

**Solution**: Dynamic imports with custom hook for URL-based modal state:

```typescript
const useModal = (modalKey: string) => {
  const router = useRouter();
  const searchParams = useSearchParams();

  const isOpen = searchParams.get("modal") === modalKey;

  const openModal = () => {
    router.push(`?modal=${modalKey}`);
  };

  return { isOpen, openModal, closeModal };
};
```
