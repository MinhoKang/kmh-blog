## 프로젝트 개요

나우즈 케어(원격)팀의 **종합 학생 관리 프로그램**을 10개월간 개발하여 성공적으로 출시했다.

실시간 채팅과 할 일 부여/확인을 핵심 기능으로 하는 원격 학습 관리 시스템이다.

## 핵심 기능

### 1. 실시간 채팅 시스템

**개발 기간**: 2024.08 ~ 2024.12

Socket.io 기반의 카카오톡 수준 채팅 시스템을 구현했다.

**주요 기능**:

- 실시간 메시지 전송/수신
- 파일 업로드 및 공유
- 메시지 검색, 삭제, 답장
- 양방향 무한 스크롤
- 크로스탭 동기화 (BroadcastChannel API)

### 2. 할 일 관리 시스템

**개발 기간**: 2025.03

react-hook-form + zod 기반의 체계적인 할 일 관리 시스템이다.

**주요 기능**:

- 할 일 CRUD (생성, 수정, 삭제)
- 카테고리별 맞춤 양식 (등교, 자기계발, 취침 등)
- 용돈 금액 및 인증 방법 설정
- 주소 검색 기능
- ModalForm + useModal 훅으로 쿼리 파라미터 상태 관리

### 3. 대시보드

**개발 기간**: 2025.02

학생 관리를 위한 종합 대시보드를 개발했다.

**주요 기능**:

- 학생 할 일 달성도 시각화
- 매니저 할 일 관리
- 학생별 개별 회의록 작성
- 메모 및 특이사항 관리

### 4. 등록 관리 (어드민)

**개발 기간**: 2025.04 ~ 2025.05

학생 및 학부모 등록을 관리하는 어드민 페이지다.

**주요 기능**:

- 등록 상태 관리 (미등록, 대기, 완료, 만료)
- 가족 구성원 연결 관리
- 재사용 가능한 Table + Tab 컴포넌트 활용

## 기술적 도전과 해결

### 1. Next.js 캐싱 최적화

**문제**: App Router에서 쿼리스트링 변경 시마다 서버 컴포넌트 재실행으로 인한 성능 저하

**원인 분석**:

- Page Router: 클라이언트 사이드 변경
- App Router: URL 변경 시 새로운 요청으로 간주
- QueryClient 인스턴스가 매번 새로 생성되어 캐싱 무효화

**해결책**:

- Next.js 캐싱 설정 수정 및 개선
- tanstack-query의 queryClient.prefetchQuery 전략 최적화
- 불필요한 API 요청 제거

### 2. 성능 최적화

**문제**: 채팅 페이지의 약 10개 모달로 인한 초기 로딩 지연

**해결책**:

- 다이나믹 임포트 적용으로 필요 시점에 모달 로드
- 번들 크기 최적화
- 사용자 체감 성능 향상

### 3. 보안 강화

**대응**: Next.js CVE-2025-29927 보안 취약점 발견 즉시 15.2.3 버전 업데이트 적용

## 결론

NOWZ-RP 프로젝트는 **10개월간의 대규모 개발 경험**을 통해 풀스택 개발자로서의 역량을 크게 성장시킨 프로젝트다.

단순한 기능 개발을 넘어서 **성능 최적화, 보안 강화, 품질 관리**까지 프로덕션 환경에서 요구되는 모든 측면을 경험했다.

특히 **Next.js App Router의 캐싱 최적화**와 **실시간 시스템 구축**을 통해 현대 웹 개발의 핵심 기술들을 실무에 적용할 수 있었으며, 앞으로도 **사용자 중심의 안정적인 서비스** 개발에 집중하고자 한다.
