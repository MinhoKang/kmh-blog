## 프로젝트 개요

"**뭘 먹을지 고민되는 순간**"을 해결하기 위해 개발한 **위치 기반 랜덤 맛집 추천 모바일 앱**이다.

사용자 현재 위치를 기준으로 원하는 거리 내의 음식점, 카페, 술집을 랜덤으로 추천해주는 서비스로 **구글 플레이스토어에 실제 출시**한 개인 사이드 프로젝트다.

## 핵심 기능

### 1. 위치 기반 랜덤 추천

**사용자 현재 위치**를 기준으로 설정한 거리 내의 맛집을 랜덤으로 추천하는 핵심 기능이다.

**주요 기능**:

- 현재 위치 자동 감지 (Expo Location)
- 거리 설정 (500m ~ 5km)
- 카테고리 선택 (식당, 카페, 술집)
- 랜덤 맛집 추천 알고리즘

### 2. 지도 연동 서비스

추천된 맛집의 **위치 정보와 상세 정보**를 다양한 방식으로 제공한다.

## 기술적 도전과 해결

### 1. 카카오 API 검색 결과 제한 우회

**문제**: 카카오 Places API의 검색 결과가 45개로 제한되어 넓은 범위 검색 시 선택지 부족

**상황 분석**:

- 2km 이상 범위에서는 45개 결과로는 다양성 부족
- 사용자가 원하는 거리 내 모든 맛집을 찾기 어려움

## 학습한 점

### 기술적 성장

- **위치 기반 서비스**: GPS, 지도 API를 활용한 모바일 서비스 개발 전문성
- **API 최적화**: 제한된 외부 API를 효율적으로 활용하는 창의적 솔루션 개발
- **React Native 심화**: Expo 생태계와 네이티브 기능 통합 경험
- **광고 플랫폼**: Google Ads 연동 및 모바일 앱 수익화 모델 구현

### 프로덕트 개발

- **전 과정 경험**: 아이디어 기획부터 개발, 배포, 운영까지 1인 개발자 경험
- **사용자 중심 설계**: 실제 사용자의 니즈를 해결하는 서비스 기획 및 구현
- **앱 스토어 정책**: 구글 플레이 정책 이해 및 심사 과정 경험

## 결론

"오늘 뭐 먹지" 앱 개발을 통해 **개인 사이드 프로젝트의 전 과정**을 경험했다.

아이디어 기획부터 기술적 문제 해결, 실제 앱스토어 배포까지 **1인 개발자로서의 역량**을 크게 성장시킨 의미 있는 프로젝트다.

특히 **카카오 API 제한을 창의적으로 우회**하여 서비스 품질을 5배 향상시킨 경험은 제한된 리소스 환경에서의 **문제 해결 능력**을 보여주는 사례다.

앞으로도 **사용자의 일상 문제를 해결**하는 실용적인 서비스를 지속 개발하며, **기술과 비즈니스를 연결**하는 프로덕트 개발자로 성장하고자 한다.
