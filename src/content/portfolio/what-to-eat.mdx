## 프로젝트 개요

위치 기반 랜덤 맛집 추천 모바일 앱
사용자 현재 위치를 기준으로 원하는 거리 내의 음식점, 카페, 술집을 랜덤으로 추천해주는 서비스

앱 개발부터 구글 플레이스토어 배포까지 경험

### 개발 기간

2025년 6월~

### 기술 스택

- **프레임워크**: React Native + Expo (SDK 53) + React 19
- **언어**: TypeScript
- **스타일링**: NativeWind, clsx
- **지도/위치**: 카카오 지도 API, Expo Location

### 기능

- 현재 위치에서 설정한 거리 내의 설정한 카테고리의 식당, 카페 추천
- 카카오맵, 네이버 지도로 해당 식당 보기
- 식당 정보 공유 기능
- 본 식당 목록
- Google Ads 적용
