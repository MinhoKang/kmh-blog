## 프로젝트 개요

나우즈 케어(원격) 고객인 **학부모와 학생이 사용하는 모바일 앱**을 React Native + Expo로 개발하여 **앱스토어와 구글 플레이에 성공적으로 출시**했다.

실시간 채팅과 학생의 하루를 보고하는 일일 관찰 페이지가 핵심 기능이다.

### 2. 토큰 관리 시스템

**개발 기간**: 2025.03

Axios 인터셉터 기반의 **JWT 토큰 자동 갱신 시스템**을 구축했다.

**주요 기능**:

- 토큰 만료 시 자동 갱신 후 원본 요청 재시도
- 중복 갱신 요청 방지 (전역 Promise 플래그)
- SecureStore를 통한 안전한 토큰 저장
- 갱신 대기 중인 요청들의 Queue 관리

### 3. UI 컴포넌트 개발

**개발 기간**: 2025.02 ~ 2025.03

@gorhom/bottom-sheet 라이브러리를 활용한 **재사용 가능한 컴포넌트 시스템**을 구축했다.

### 4. Expo Updates 관리

**개발 기간**: 2025.08

**EAS-Update**를 활용한 앱 업데이트 관리 시스템을 구현했다.

**주요 기능**:

- 사용자에게 업데이트 알림 모달 제공
- 업데이트 완료 후 자동 앱 재시작 (reloadAsync)
- 재심사 없는 빠른 배포 지원

### 2. 확장 가능한 폼 구조 설계

**문제**: 향후 질문 항목 추가/변경이 빈번할 것으로 예상되는 비즈니스 요구사항

**해결책**: enum 기반 타입 정의와 동적 폼 생성 로직으로 **코드 수정 없이 설정만으로 폼 구조 변경 가능**한 아키텍처 구현

### 3. WebView 위치 조회 문제

**문제**: react-native-webview로 카카오 지도 사용 시 위치 조회 기능 미작동

**해결책**: WebView의 source 속성 중 **baseUrl을 수정**하여 위치 조회 기능 정상 작동

### 4. 토큰 갱신 중복 요청 방지

**문제**: 토큰 갱신 시 중복 요청으로 인한 서버 부하 및 갱신 실패
