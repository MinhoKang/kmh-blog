## The Technical Challenges

### Challenge #1: Kakao API Limitations

The biggest technical hurdle was Kakao's API limit of 45 search results per query. For larger search radii, this meant users might miss great nearby options.

**Problem**: User sets 3km radius â†’ Only gets 45 restaurants max â†’ Missing potentially better options

**Solution**: Implemented multi-zone search strategy

```typescript
// Multi-zone search for areas > 2km
const performMultiZoneSearch = async (center: Location, radius: number) => {
  if (radius <= 2000) {
    return performSingleSearch(center, radius);
  }

  // Divide search area into 4 quadrants
  const quadrants = [
    { lat: center.lat + offset, lng: center.lng + offset }, // NE
    { lat: center.lat + offset, lng: center.lng - offset }, // NW
    { lat: center.lat - offset, lng: center.lng + offset }, // SE
    { lat: center.lat - offset, lng: center.lng - offset }, // SW
  ];

  const allResults = await Promise.all(
    quadrants.map((point) => performSingleSearch(point, radius / 2))
  );

  // Remove duplicates and sort by distance
  return removeDuplicatesAndSort(allResults.flat(), center);
};
```

This increased the effective search pool from 45 to up to 225 restaurants.

        setUpdateAvailable(true);
        showUpdateModal();
      }

    }

    await Updates.fetchUpdateAsync();
    await Updates.reloadAsync();
  };

## Technical Deep-Dive: Random Selection Algorithm

The core of the app is smart randomization. Can't just use `Math.random()` - need to consider distance, ratings, and user preferences:

```typescript
interface WeightedSelection {
  restaurant: Restaurant;
  weight: number;
}

const calculateWeight = (restaurant: Restaurant, userLocation: Location) => {
  const distanceWeight = Math.max(0, 1 - restaurant.distance / MAX_DISTANCE);
  const ratingWeight = restaurant.rating / 5;
  const categoryWeight = userPreferences.includes(restaurant.category)
    ? 1.2
    : 1;

  return distanceWeight * 0.4 + ratingWeight * 0.4 + categoryWeight * 0.2;
};

const selectRandomRestaurant = (restaurants: Restaurant[]): Restaurant => {
  const weightedOptions: WeightedSelection[] = restaurants.map(
    (restaurant) => ({
      restaurant,
      weight: calculateWeight(restaurant, userLocation),
    })
  );

  const totalWeight = weightedOptions.reduce(
    (sum, option) => sum + option.weight,
    0
  );
  let randomValue = Math.random() * totalWeight;

  for (const option of weightedOptions) {
    randomValue -= option.weight;
    if (randomValue <= 0) {
      return option.restaurant;
    }
  }

  return weightedOptions[0].restaurant; // Fallback
};
```
