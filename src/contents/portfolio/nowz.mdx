## 프로젝트 개요

학생의 변화와 성장을 위해 대시보드, 수업 관리, 과제 관리, 학생 분석 레포트, 일정 관리 등 **종합적인 학생 관리 기능**을 제공하는 시스템

동일 회사에서 타부서 팀장으로 있다가 개발팀으로 넘어온 뒤 맡은 프로젝트
첫 개발부터 시작하지는 않았고, 서비스 시작과 함께 유지보수, 기능 개발

---

## 기술 스택

- 프레임워크: React 18 + Vite
- 언어: TypeScript
- 상태 관리: tanstack-query, recoil
- 스타일링: tailwind
- 통신: axios
- 폼 유효성 검사: react-hook-form + zod
- 컴포넌트 개발: Storybook
- 모니터링/에러: sentry

## 내가 한 역할

### 1. 상태 관리 시스템 개선

**기존 Recoil로 관리되던 전역 상태**를 Tanstack-Query 기반으로 전면 재구축

### 2. Storybook 기반 디자인 시스템 구축

컴포넌트화 되어있지 않았던 시스템을 **Storybook을 이용안 디자인 시스템 구축 및 컴포넌트 재사용**

### 3. 검색 에이전트 “Captain” 개발

- 사용자의 질문에 데이터베이스 기반으로 답하는 Captain을 개발
- ReadableStream API를 활용하여 청크 단위 데이터을 처리
- TextDecoder를 통해 바이너리 데이터 디코딩을 경험
- 타이핑 효과가 있는 실시간 메시지 표시 기능을 구현

### 4. 학생 분석 레포트 시스템

**Chart.js를 활용**하여 학생들의 학습 데이터를 시각화하는 분석 레포트 페이지(Grade Card) 및 주간 보고서 페이지를 개발

### 6. 학생 상세 페이지 TOC(Table Of Content) 개발

- IntersectionObserver를 이용하여 id를 찾아 현재 보이는 메뉴 하이라이팅

```tsx
useEffect(() => {
  const observer = new IntersectionObserver(
    (entries) => {
      entries.forEach((entry) => {
        if (entry.isIntersecting) {
          const index = TAB_LIST.findIndex(
            (tab) => tab.href === entry.target.id
          );
          if (index !== -1) {
            setNowMenu(index);  // 스크롤 위치에 따라 자동 하이라이트
          }
        }
      });
    },
    { threshold: 1 }  // 100% 보일 때만 트리거
  );
```

- 메뉴를 클릭할 경우 해당 id를 가진 메뉴로 이동

```tsx
const handleClick = (index: number, href: string) => {
  setNowMenu(index); // 상태 업데이트
  const targetElement = document.getElementById(href);
  if (targetElement) {
    targetElement.scrollIntoView({ behavior: "smooth" }); // 부드러운 스크롤
  }
};
```
