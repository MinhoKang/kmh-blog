## 프로젝트 개요

상담팀이 사용하는 **통합 상담 관리 프로그램**을 React 19 + Vite 기반으로 개발.

상담 예약, 변경, 취소부터 상담 기록 관리까지 **상담 업무의 전 과정을 디지털화**한 시스템.

현재 **지속적 개발 및 유지 보수** 중

---

## 기술 스택

- **프레임워크**: React 19 + Vite
- **언어**: TypeScript
- **상태 관리**: tanstack-query
- **스타일링**: tailwind, clsx
- **폼 유효성 검사**: react-hook-form + zod
- **통신**: ky
